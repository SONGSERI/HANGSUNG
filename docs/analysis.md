# Possible Analysis Using Parsed Data Structure

본 항목은 현재 정의된 데이터 구조(LOT, MACHINE_RUN 및 요약 테이블)를 기반으로  
수행 가능한 분석 범위를 정리한 것이다.  
실제 분석 과제는 데이터 분포 및 품질 검토 후 선정한다.

---

## 1. 설비 가동 현황 분석

### 활용 테이블
- MACHINE_TIME_SUMMARY
- MACHINE_RUN

### 분석 내용
- 설비별 가동 시간 / 유휴 시간 / 정지 시간 비율
- 실제 가동 시간 대비 생산 시간 비교
- Lot별 설비 가동 패턴 비교

### 목적
- 설비 운영 상태 가시화
- 비효율 시간 영역 식별

---

## 2. 정지 유형 기반 운영 분석

### 활용 테이블
- MACHINE_STOP_SUMMARY
- MACHINE_RUN

### 분석 내용
- 정지 원인별 시간 및 발생 빈도 비교
- 설비별 정지 유형 분포
- Lot별 정지 패턴 차이

### 목적
- 주요 정지 유형 식별
- 운영 관점의 개선 대상 영역 도출

---

## 3. 생산 실적 및 처리량 분석

### 활용 테이블
- PRODUCTION_QUALITY_SUMMARY
- LOT

### 분석 내용
- Lot별 Board / Module 처리량
- 설비별 생산 처리량 비교
- 시간 대비 생산량 분석

### 목적
- 생산 실적 가시화
- 설비 처리 능력 비교

---

## 4. 품질 지표 기초 분석

### 활용 테이블
- PRODUCTION_QUALITY_SUMMARY

### 분석 내용
- Pickup Miss / Recognition Miss 발생 비율
- 총 Pickup 대비 정상 Mount 비율
- Lot 및 설비별 품질 지표 비교

### 목적
- 품질 편차 식별
- 품질 관련 이상 Lot 또는 설비 탐색

---

## 5. 공정 대기 및 병목 분석

### 활용 테이블
- MACHINE_TIME_SUMMARY

### 분석 내용
- 전공정 / 후공정 대기 시간 비중
- 설비별 대기 시간 분포
- Lot별 대기 패턴 비교

### 목적
- 공정 병목 후보 영역 식별
- 라인 밸런스 검토 기초 자료 확보

---

## 6. Lot 단위 운영 특성 비교

### 활용 테이블
- LOT
- MACHINE_RUN
- 요약 테이블 전체

### 분석 내용
- 동일 제품 Lot 간 가동·정지·품질 특성 비교
- 제품 Revision 또는 Stage별 차이 분석

### 목적
- Lot 조건 차이에 따른 운영 특성 파악
- 표준 Lot 운전 패턴 정의 기초 자료

---

## 7. 추이 및 비교 분석 (시간 기준)

### 활용 테이블
- MACHINE_RUN (collect_time)

### 분석 내용
- 시간 흐름에 따른 설비 가동/정지 변화
- 특정 시점 전후 운영 특성 비교

### 목적
- 운영 상태 변화 감지
- 개선 활동 전·후 비교 기준 확보

---

## 8. 관리 및 보고 지표 구성

### 활용 테이블
- 요약 테이블 전체

### 분석 내용
- 설비별 핵심 운영 지표(KPI) 산출
- Lot 단위 운영 요약 리포트 생성

### 목적
- 현황 보고
- 의사결정 참고 자료 제공

---

## 9. 분석 범위 제한 사항

### 현재 구조로 불가능한 분석
- 개별 이벤트 타임라인 분석
- Feeder / Nozzle 단위 원인 추적
- 실시간 이상 감지
- 정밀 원인 규명 및 예측 분석

해당 분석은 이벤트 로그 또는 추가 데이터 수집이 필요하다.
